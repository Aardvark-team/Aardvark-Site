import http from std
import file from std

function handle(request) {
    if std.http.path(request) {
        return new std.File(
            std.http.getPath(request)
        ).readAll()
    }
    let url = std.http.url(request)
    if url = “mysite.com”
        return new std.File(“index.html”).get
}

Easier than Python

The language requires no type annotations, and uses familiar {/} syntax with high levels of inference so that your code can be clean and minimalist. It doesn’t require (/) and the constructs of the language are easy to use, with minimal syntax.

Faster than C

The language uses static typing and compile-time validation to ensure performance matching or oftentimes exceeding low-level languages like C by using static memory freeing under the hood and avoiding runtime solutions such as ARC or GC.

Safer than Rust

The language uses error values instead of try-throw-catch, and doesn’t have an unsafe system or raw pointers. Moreover, it ensure that no performance is lost without including undefined behaviour, which it does through compile-time validation.
